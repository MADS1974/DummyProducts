## 🛒 DummyProducts App


Um aplicativo Android para consulta de catálogo de produtos consumindo a API REST DummyJSON desenvolvido em Kotlin.

O objetivo principal é demonstrar o consumo de APIs externas, processamento de JSON com GSON e carregamento dinâmico 

de imagens utilizando a biblioteca Volley.Este projeto foi criado para fins acadêmicos, com foco em:

#### •Comunicação assíncrona com redes via HTTP/REST

#### •Processamento de dados estruturados com GSON

#### •Carregamento de imagens em segundo plano com ImageRequest

#### •Implementação do padrão Singleton para filas de requisições

#### •Uso de RecyclerView para exibição de listas dinâmicas de imagens

#### •Interação dinâmica com Spinner e atualização de UI em tempo real


## 🚀 Funcionalidades

### ✔️ Listagem de Produtos: 
Busca uma lista completa de produtos da API DummyJSON. 

📊 Seleção Dinâmica: 
Spinner para escolher entre diferentes produtos. 

🖼️ Galeria de Imagens: 
Exibe todas as fotos do produto selecionado em um RecyclerView. 

💰 Detalhes em Tempo Real: 
Exibe o preço e a descrição detalhada assim que o produto é selecionado. 

⚡ Performance Otimizada: 
Uso de fila de requisições (Singleton) para evitar sobrecarga de memória. 

🧹 Gerenciamento de Estado: 
Limpeza automática da galeria anterior ao trocar de produto.


## 🛠 Tecnologias Utilizadas

#### Kotlin — Linguagem principal.
#### ViewBinding — Acesso seguro e tipado aos componentes da interface.
#### Volley — Biblioteca do Google para gerenciamento de requisições de rede e imagens.
#### GSON — Conversão automática de JSON para objetos Kotlin (POJO).
#### RecyclerView — Exibição eficiente da galeria de fotos.
#### Spinner — Seletor de produtos integrado ao modelo de dados.
#### Material Design — Componentes visuais modernos e Toolbars personalizadas.
